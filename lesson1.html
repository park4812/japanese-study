문법 ① ～です

「～です」 = 한국어의 「~입니다」「~예요/이에요」
사람이나 물건에 대해 설명할 때 사용
문형: 名詞（めいし）+ です 명사 + 입니다

긍정肯定文（こうていぶん）

  • 私わたしは韓国人かんこくじんです。
    저는 한국인입니다.
  • 私わたしは会社員かいしゃいんです。
    저는 회사원입니다.
  • 私わたしは旅行者りょこうしゃです。
    저는 여행객입니다.

부정否定文（ひていぶん） — 名詞 + ではありません

  • 私わたしは日本人にほんじんではありません。
    저는 일본인이 아닙니다.
  • 私わたしは学生がくせいではありません。
    저는 학생이 아닙니다.

의문疑問文（ぎもんぶん） — 名詞 + ですか？

  • 韓国人かんこくじんですか？
    한국인입니까?
  • 旅行者りょこうしゃですか？
    여행객입니까?

대답答（こた）え方（かた）

  • はい、韓国人かんこくじんです。
    네, 한국인입니다.
  • いいえ、韓国人かんこくじんではありません。
    아니요, 한국인이 아닙니다.

문법 ② ～から来きました

"어디에서 왔는지"를 말하는 표현 = 한국어의 「~에서 왔습니다」
문형: 場所（ばしょ）+ から + 来（き）ました 장소 + 에서 + 왔습니다

긍정肯定文

  • 韓国かんこくから来きました。
    한국에서 왔습니다.
  • ソウルから来きました。
    서울에서 왔습니다.
  • 釜山ぷさんから来きました。
    부산에서 왔습니다.

부정否定文 — 場所 + から + 来（き）ていません

  • 韓国かんこくから来きていません。
    한국에서 오지 않았습니다.
  • 東京とうきょうから来きていません。
    도쿄에서 오지 않았습니다.

의문疑問文 — 場所 + から + 来ましたか？

  • 韓国かんこくから来きましたか？
    한국에서 오셨습니까?
  • ソウルから来きましたか？
    서울에서 오셨습니까?

질문핵심 질문과 대답

  • どこから来きましたか？
    어디에서 오셨습니까?
  • 韓国かんこくから来きました。／ソウルから来きました。
    한국에서 왔습니다. / 서울에서 왔습니다.

문법 ③ 趣味しゅみは ～ です

취미에 대해 이야기할 때 사용 = 한국어의 「취미는 ~입니다」
문형: 趣味（しゅみ）は + 名詞 + です 취미는 + 명사 + 입니다

「は」의 역할: 문장의 주제를 나타냄 — 한국어 조사 「은/는」과 비슷 (발음은 'わ wa')

긍정肯定文

  • 趣味しゅみは旅行りょこうです。
    취미는 여행입니다.
  • 趣味しゅみは映画えいがです。
    취미는 영화입니다.
  • 趣味しゅみは料理りょうりです。
    취미는 요리입니다.

부정否定文

  • 趣味しゅみはスポーツではありません。
    취미는 스포츠가 아닙니다.
  • 趣味しゅみはゲームではありません。
    취미는 게임이 아닙니다.

의문疑問文

  • 趣味しゅみは旅行りょこうですか？
    취미는 여행입니까?
  • 趣味しゅみは料理りょうりですか？
    취미는 요리입니까?

질문핵심 질문과 대답

  • 趣味しゅみは何なんですか？
    취미가 무엇입니까?
  • 趣味しゅみは旅行りょこうです。／趣味しゅみは写真しゃしんです。
    취미는 여행입니다. / 취미는 사진입니다.

자기소개 기본 패턴 基本（きほん）パターン

  1. はじめまして。
    처음 뵙겠습니다.
  2. 私わたしの名前なまえはミナです。
    제 이름은 미나입니다.
  3. 韓国かんこくから来きました。
    한국에서 왔습니다.
  4. 旅行りょこうで日本にほんに来きました。
    여행으로 일본에 왔습니다.
  5. 趣味しゅみは旅行りょこうです。
    취미는 여행입니다.
  6. よろしくお願ねがいします。
    잘 부탁드립니다.

모델 회화 モデル会話（かいわ）

A はじめまして。私わたしの名前なまえはミナです。韓国かんこくから来きました。趣味しゅみは旅行りょこうです。よろしくお願ねがいします。
처음 뵙겠습니다. 제 이름은 미나입니다. 한국에서 왔습니다. 취미는 여행입니다. 잘 부탁드립니다.
B はじめまして。私わたしはたろうです。日本にほんへようこそ。
처음 뵙겠습니다. 저는 타로입니다. 일본에 오신 것을 환영합니다.

질문해 보기 質問（しつもん）してみよう

  • お名前なまえは何なんですか？
    이름이 무엇입니까?
  • どこから来きましたか？
    어디에서 오셨습니까?
  • 趣味しゅみは何なんですか？
    취미가 무엇입니까?
  • 日本にほんのどこへ行いきますか？
    일본의 어디에 가십니까?
